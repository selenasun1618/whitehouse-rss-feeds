# White House RSS Feeds

White House Briefings & Statements RSS Feed Generator

Scrapes https://www.whitehouse.gov/briefings-statements/ and generates a valid RSS 2.0 feed.

## Installation

Install the required dependencies:

```bash
pip3 install -r requirements.txt
```

## Usage

Run the scraper:

Or with Python 3:

```bash
python3 whitehouse_rss_scraper.py
```

## Output

The script will create `whitehouse_briefings.xml` in the current directory containing the RSS feed.
